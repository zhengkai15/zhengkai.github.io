网站正在建立 内容如有雷同马上撤回

Mar 10, 2021 #poem

轻轻的你走了，真如我悄悄的更在后面，保护你的安全。 

brew install hugo # 安装hugo，其他平台类似，参考：<https://gohugo.io/getting-started/installing/>

hugo new site myblog # 创建一个新站点myblog，会新建一个文件夹myblog，并在该文件夹中生成基础的网页配置模块。
